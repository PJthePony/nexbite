Today Column Design Principles
  • Wider: Today gets ~1.3x flex ratio vs. other days
  • Border accent: 2px primary-color border + subtle box-shadow glow
  • Header emphasis: Primary color title + "Today" badge
  • Rich cards: Full task cards with checkbox, tags, bite indicator, hover actions
  • Workstream labels: Visible inside today column to group tasks
  • Other days: Compact cards (title only), muted header color, thinner borders
  • Background: Slightly warm/tinted bg distinguishes today from the grid

Quick Reference: How to Choose

Option Character Best For
Font A — Inter Neutral, invisible, lets content lead Maximum clarity, Google-style restraint
Font B — Plus Jakarta Sans Warm, modern, a touch of personality Approachable professional, Airbnb-level polish
Font C — IBM Plex Sans Authoritative, technical, precise Data-dense views, developer tools aesthetic
Palette A — Warm Blue Approachable, grounded, Asana warmth All-day comfort, balanced professionalism
Palette B — Cool Indigo Sharp, focused, Linear precision Developer-first, strong visual hierarchy
Palette C — Muted Teal Calm, organic, easy on the eyes Reduced eye strain, zen productivity
To decide: Say something like "I want Font B with Palette A" and we will apply that combination across the entire app.
